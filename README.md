# Marketing Funnel & Campaign Performance Analysis
Marketing funnel and campaign performance analysis using SQL and Power BI


##  Project Overview
This project analyzes marketing funnel performance and campaign effectiveness to identify conversion drop-offs and optimize marketing spend.

##  Business Problem
Marketing campaigns are generating traffic, but conversions are inconsistent. The goal is to:
- Identify funnel stages with maximum drop-off
- Evaluate campaign and channel performance
- Optimize marketing ROI

##  Tools Used
- SQL (MySQL)
- Power BI

##  Dataset
Marketing campaign performance dataset including impressions, clicks, leads, conversions, cost, and revenue.

##  Key Analysis
- Funnel analysis from impressions to conversions
- Campaign-wise and channel-wise performance analysis
- Cost per lead, CAC, and ROI calculation

##  Key Insights
- Certain campaigns generate high clicks but low conversions
- Email campaigns show higher ROI compared to paid ads
- Major funnel drop observed at the click-to-lead stage

##  Recommendations
- Optimize landing pages to improve lead conversion
- Reallocate budget to high-ROI campaigns
- Pause or redesign low-performing campaigns

##  Outcome
This analysis supports data-driven marketing decisions and improves campaign efficiency.
